Gestión SST para Minería

Seguridad y Salud en el Trabajo

Aplicación para la gestión de seguridad y salud en el trabajo, dirigida al sector minero. Desarrollada con Laravel y SQL Server 2014, ofrece análisis de datos, geolocalización con Google Maps, y validación de APIs con Postman. Incluye envío de mensajes por correo masivo para reportar actos inseguros, condiciones e incidentes a través de una aplicación APK, optimizando la gestión y seguridad minera.
